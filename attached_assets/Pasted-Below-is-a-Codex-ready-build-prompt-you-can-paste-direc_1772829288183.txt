# Codex Build Prompt

You are building a **territory intelligence automation tool** for biotechnology sales prospecting.

The tool must automatically discover **biotech, pharmaceutical, and CRO companies** located in the following territory:

San Francisco → Mountain View, California.

The system will identify companies, remove duplicates, validate their existence, confirm they are within the territory, and output qualified companies to a **Google Sheet**.

The tool must be written using:

* Python (backend)
* React (frontend dashboard)
* OpenAI API
* Perplexity API
* Gemini API
* Google Sheets API
* OpenStreetMap Nominatim geocoding API

The system must **NOT rely on any paid databases** such as Crunchbase or PitchBook.

The application should be structured as a **modular pipeline**.

---

# Core Workflow

1. Load two tabs from Google Sheets:

Tab 1: `excludedCompanies`
Tab 2: `startingList`

These contain companies that must not be searched.

2. Validate companies in `startingList`

For each company:

* Confirm the website exists (HTTP check)
* Confirm the company still exists using Perplexity
* Confirm the company is located within the territory

Companies that fail validation are removed.

3. Create master exclusion database:

```
dontSearch = excludedCompanies + startingList
```

4. Discovery Loop

Perplexity and Gemini will search for companies in the territory.

Each iteration must return **5 companies not in dontSearch**.

Example prompt to LLM:

"List 5 biotechnology, pharmaceutical, or CRO companies located in the Bay Area between San Francisco and Mountain View that are not in the following list: [dontSearch]. Return only company name and website."

5. Deduplicate results

Use fuzzy matching.

Similarity threshold: **92%**

Use RapidFuzz.

6. Territory validation

Each discovered company must:

* Have an address
* Be geocoded using OpenStreetMap Nominatim
* Fall within bounding coordinates

Bounding box:

```
lat_min = 37.35
lat_max = 37.81
lon_min = -122.55
lon_max = -121.85
```

7. Update dontSearch

Add validated companies.

8. Stop Condition

Stop when **3 consecutive iterations produce no new companies**.

9. Output

Create a new tab in the Google Sheet:

```
prospectDiscovery
```

Columns:

```
Company Name
Company Website
Overview
Notes
```

Overview must be a **5 sentence company description** generated by OpenAI.

# Geocoder Module

Use OpenStreetMap Nominatim.

Example:

```python
def geocode_address(address):

    url = "https://nominatim.openstreetmap.org/search"

    params = {
        "q": address,
        "format": "json"
    }

    response = requests.get(url, params=params)
    data = response.json()

    if data:
        return float(data[0]["lat"]), float(data[0]["lon"])

    return None
```

Territory check:

```python
def in_territory(lat, lon):

    lat_min = 37.35
    lat_max = 37.81
    lon_min = -122.55
    lon_max = -121.85

    return (
        lat_min <= lat <= lat_max and
        lon_min <= lon <= lon_max
    )
```

# Discovery Loop Logic

Pseudo code:

```python
zero_rounds = 0

while zero_rounds < 3:

    results = search_for_companies()

    new_companies = remove_duplicates(results)

    if new_companies:
        add_to_dont_search(new_companies)
        zero_rounds = 0
    else:
        zero_rounds += 1
```
